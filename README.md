# Collège Maendeleo - Website

**Le Paradis du Savoir** | Professional School Website

## 📋 Overview

A modern, professional website for Collège Maendeleo featuring three distinct educational sections (Maternelle, Primaire, Secondaire) and 12+ academic options.

## 🗂️ Project Structure

```
school-website/
│
├── index.html          # Homepage with hero, features, sections overview
├── about.html          # About page with leadership and stats
├── sections.html       # Detailed sections (Maternelle, Primaire, Secondaire)
├── options.html        # 12+ academic options showcase
├── contact.html        # Contact form and information
│
├── css/
│   ├── style.css           # Main stylesheet with design system
│   └── header-footer.css   # Navigation and footer styles
│
├── js/
│   └── main.js            # All interactive functionality
│
└── images/
    └── (Place your school images here)
```

## 🚀 Getting Started

### Installation
1. Extract all files maintaining the folder structure
2. Add your school images to the `images/` folder
3. Open `index.html` in a web browser

### Image Requirements
- Logo: `images/logo.png` (recommended 200x200px)
- Leadership photos: `images/gestionaire.jpg` and others
- Replace placeholder images with actual school photos

### Customization

#### Colors
Edit CSS variables in `css/style.css`:
```css
:root {
    --color-primary: #1a5fb4;      /* Primary blue */
    --color-secondary: #e06c75;    /* Coral/pink */
    --color-accent: #61afef;       /* Light blue */
    /* ... more colors */
}
```
